# DBOS Knowledge Index

## Purpose

This index provides a single navigation point for the current DBOS knowledge packages.

| Knowledge package | Purpose |
|---|---|
| Aim, Goal, Rewrite | Reviews an email against its intended purpose and produces a stronger rewritten version. |
| Bionic Reader | Converts supplied text into Bionic Reading format without changing the original wording or structure. |
| Core Knowledge | Holds shared criteria, KSBs, assessment resources and common reference material. |
| Council of Dave | Provides structured challenge, review and multi-perspective decision support. |
| Direct Questions | Builds merged, holistic assessment questions aligned to criteria and self-evaluation modules. |
| Duck Hunt | Supports evidence-based assessment decisions, personalised feedback and criteria mapping. |
| Hodder | Holds Level 5 adult care unit learning and reference resources. |
| Learning Plan Planner | Produces realistic schedules for apprenticeship assessments, observations and milestones. |
| Legislation | Maintains live links to legislation, regulations and authoritative guidance. |
| Teaching and Learning Review | Produces accurate teaching and learning records focused on active learning, impact and next steps. |

## Maintenance

Update this index whenever a knowledge package is added, renamed, archived or materially repurposed.
