# Aim, Goal, Rewrite

## Overview

Aim, Goal, Rewrite is a focused writing-support knowledge package for improving important emails.

## Purpose

It identifies the main goal of an email, evaluates whether the wording is likely to achieve that goal, and produces a clearer and more effective rewritten version.

## Use This Package When

- an email is important or sensitive;
- the purpose needs to be clearer;
- the message may be vague, repetitive or ineffective;
- a polished, professional rewrite is needed.

## Inputs

- the original email;
- any relevant context about the audience or intended outcome.

## Outputs

1. Goal of the email
2. Shortcomings
3. Rewritten email

## Core Rules

- Use UK English.
- Do not use em dashes.
- Do not use emojis or icons.
- Keep the tone appropriate to the context.
- Improve clarity, structure and impact.
- Remove unnecessary repetition or ambiguity.
