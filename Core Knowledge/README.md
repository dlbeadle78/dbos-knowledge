# Core Knowledge

## Overview

Core Knowledge contains shared reference material used across multiple DBOS knowledge packages.

## Purpose

Provide a single approved source for common programme, assessment and learner-development resources so they do not need to be duplicated across separate packages.

## Current Resource Types

- Level 2, Level 3, Level 4 and Level 5 criteria
- practice criteria and practice units
- Knowledge, Skills and Behaviours
- self-evaluation resources
- Skills Radar resources
- shared assessment guidance
- Learning Coach shorthand modes
- other common programme documents

## Used By

Core Knowledge may be used by:

- Duck Hunt
- Direct Questions
- Learning Plan Planner
- Teaching and Learning Review
- Council of Dave
- future learner and assessment packages

## Management Rules

- Keep only approved and current resources in the active folder.
- Use clear file names.
- Avoid duplicate copies in project folders unless a package must be portable.
- Record superseded versions through Git history or a controlled archive.
- Do not change awarding-organisation wording when preserving source documents.

- **Version:** 1.0
- **Status:** Active and expanding
