# DBOS Knowledge Index (Updated)

## Repository Purpose

The DBOS Knowledge Repository contains the approved operational knowledge used throughout DBOS.

It complements the DBOS Standards repository by providing the practical knowledge, reference material and project-specific resources that support day-to-day work.

---

# Core Knowledge

Shared resources used across multiple knowledge packages.

| Package | Purpose |
|---------|---------|
| Core Knowledge | Shared criteria, KSBs, assessment standards, programme documents and reusable reference material. |
| Legislation | Live legislation, regulations and authoritative guidance for Health and Social Care. |
