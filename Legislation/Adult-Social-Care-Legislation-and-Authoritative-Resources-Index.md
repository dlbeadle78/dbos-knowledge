**Last link review:** 22 July 2026  
**Use:** This is a navigation index to live authoritative sources. It is not legal advice and does not replace checking the current text, amendments, commencement provisions, jurisdiction or organisation-specific policy.

## Maintenance rules for this index

1. Link to the **HTML contents page**, not a downloaded PDF, wherever possible.
2. Treat legislation.gov.uk as the authoritative source for enacted UK legislation, while checking each page's **Changes to Legislation** status.
