# Learning Plan Planner

## Overview

Learning Plan Planner creates realistic schedules for apprenticeship assessments, observations, referrals and final milestones.

## Purpose

Turn a list of outstanding activities and a final deadline into a logical, achievable learning plan.

## Primary Functions

- expand recognised abbreviations;
- sequence activities logically;
- distribute deadlines across the available period;
- allow realistic time for referrals;
- spread observations through the plan;
- preserve required gaps between observations;
- position Expert Witness Testimony after Observation 3;
- place Gateway or another final milestone at the end where appropriate.

## Inputs

- list of assessments and activities;
- final deadline;
- any fixed dates or spacing requirements;
- details of referrals or activities needing additional time.

## Output

One Markdown table using:

| Full Name | Deadline |
|---|---|

Dates use UK format: `DD/MM/YYYY`.

## Core Rules

- Use full activity names.
- Do not leave unexplained abbreviations.
- Order activities from earliest to latest.
- Keep the plan practical rather than mechanically even.
- Do not add notes or order numbers unless requested.

## Repository Contents

This folder should contain:

- Learning Plan Planner Instructions
- current abbreviation key
- approved planning examples where useful
