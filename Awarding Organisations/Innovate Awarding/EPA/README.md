# Innovate Awarding: Adult Care EPA Scope

This section is intentionally limited to the four adult-care apprenticeship standards required by DBOS:

1. Adult Care Worker — Level 2 — ST0005
2. Lead Adult Care Worker — Level 3 — ST0006
3. Lead Practitioner in Adult Care — Level 4 — ST0007
4. Leader in Adult Care — Level 5 — ST0008

The master register is `adult-care-epa-register.csv`. It records each publicly listed Innovate version, product number, status, assessment methods, timings, gateway requirements, grading and official overview URL.

## Important current-version note

Adult Care Worker v2.0 is structurally different from the earlier versions. Innovate states that there is no separate apprenticeship end-point assessment; the Level 2 Adult Social Care Certificate is the mandatory qualification and assessment route. Earlier v1.2 and v1.3 versions retain the situational judgement test and professional discussion.

## Evidence boundary

The register contains only information supported by Innovate's public catalogue and apprenticeship overview pages. Detailed grading descriptors, full KSB-to-method mappings, assessment criteria and restricted Proficient materials are not reproduced. These should only be indexed from authorised or publicly available source documents.
