# Awarding Organisations

This area contains approved source records, indexes and curated knowledge from awarding organisations used by DBOS.

## Current source families

- Innovate Awarding
- City & Guilds

## Operating rules

1. The official publisher remains the authoritative source.
2. Every resource must record its title, version, publication or update date, status, source URL and date checked.
3. Superseded material must remain identifiable and must not be silently replaced.
4. DBOS must select the version that applies to the learner's programme start date, registration date or other governing date.
5. Repository summaries must not be treated as substitutes for the full official specification where exact wording, criteria, KSBs or assessment requirements are needed.
6. Copyrighted documents should be linked or stored only where permission and licensing allow.
7. Regularly changing sources must be reviewed and their version registers updated.

## Status labels

- `current` — available for current delivery or assessment
- `future` — published but not yet effective
- `superseded` — replaced by a later version
- `withdrawn` — no longer available for new registrations or starts
- `closed` — closed by the publisher, with legacy use only where applicable
- `unverified` — recorded but not yet checked against the official source
