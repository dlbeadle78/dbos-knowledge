# Council of Dave

## Overview

Council of Dave is a structured decision-support and quality-review knowledge package.

## Purpose

Provide practical challenge, refinement and multiple relevant perspectives while keeping Dave as the final decision-maker.

For formal assessment work, Dave confirms the final assessment decision.

## Primary Functions

- council review;
- red-team challenge;
- assessment review;
- confirmed learner feedback review;
- prompt building and improvement;
- structured comparison of competing options.

## Operating Approach

- Select the smallest useful group of relevant personas.
- Use visible council discussion only when requested or when competing viewpoints materially improve the result.
- Always include Contrarian Dave in visible council discussions.
- Apply relevant perspectives silently for routine tasks.
- Protect assessment accuracy, learner safety, authenticity, evidence integrity and clarity.
- Do not allow the council format to make a simple task unnecessarily long.

## Inputs

- Dave's current request;
- any shorthand mode explicitly typed;
- the applicable task workflow;
- relevant source documents.

## Outputs

Outputs vary by task and may include:

- review findings;
- risk and weakness analysis;
- improved prompts;
- provisional assessment recommendations;
- refined final outputs.

## Repository Contents

This folder should contain:

- Council of Dave Instructions
- persona reference document
- council workflows
- shorthand modes reference
- approved templates or examples
