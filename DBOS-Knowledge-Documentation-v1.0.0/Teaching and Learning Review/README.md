# Teaching and Learning Review

## Overview

Teaching and Learning Review supports the creation of accurate records of active learning, new knowledge, skills, professional behaviours, workplace application and learner development.

## Purpose

Produce meaningful teaching and learning records rather than assessment reviews, portfolio updates or submission trackers.

## Primary Functions

- record session type and delivery;
- distinguish planned, completed, confirmed and proposed activity;
- establish starting point, aim and objectives;
- describe active learner participation;
- explain how learning was checked;
- record individualisation and support;
- connect learning to verified KSBs, the learning plan and EPA;
- capture learner comments and workplace application;
- set SMART actions;
- record OTJ learning accurately;
- identify resources and agreed support.

## Inputs

- learner and programme details;
- session plan or completed-session notes;
- topic and rationale;
- starting point;
- learning activities and checks;
- learner comments;
- verified KSB or EPA links;
- agreed actions, support and resources.

## Outputs

- planned Learning Coach session record;
- completed Teaching and Learning review;
- or both, where requested.

## Core Rules

- Do not invent learner comments, scores, KSBs, actions or completed activity.
- Do not turn the record into an assessment decision.
- Separate plans from completed learning.
- Use only verified KSB references.
- Record active participation and workplace application.
- Use natural, professional UK English.

## Repository Contents

This folder should contain:

- Teaching and Learning Review Instructions
- approved templates
- relevant session-planning resources
- example records where appropriate
