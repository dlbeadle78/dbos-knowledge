# Direct Questions

## Overview

Direct Questions creates merged, holistic assessment questions for Level 2 and Level 3 Health and Social Care apprenticeships.

## Purpose

Reduce duplication, maximise appropriate criteria coverage and prepare learners for both assessment and End Point Assessment self-evaluation.

## Primary Functions

- map supplied criteria to the correct Direct Reflection Section;
- align questions with the relevant Self Evaluation module;
- merge criteria where practice naturally overlaps;
- create clear plain-English questions;
- provide sentence starters and command-verb guidance;
- produce rubrics once questions are confirmed;
- include Skills Radar scoring where supplied.

## Inputs

- assessment criteria;
- apprenticeship level;
- Skills Radar KSB questions where applicable;
- relevant Core Knowledge sources.

## Outputs

For each merged question:

- section title;
- question;
- sentence starter;
- guidance on depth and command verbs;
- criteria mapping;
- Direct Reflection Section;
- Self Evaluation module alignment.

After confirmation, the package may also produce an assessment rubric.

## Core Rules

- Do not duplicate full Self Evaluation questions.
- Do not give model answers.
- Keep one unit per criteria-mapping line.
- Use full unit numbers.
- Use clear, accessible UK English.
- Merge only where the evidence requirements genuinely overlap.

## Repository Contents

This folder should contain:

- Direct Questions Instructions
- package-specific templates
- any approved examples
- references to relevant Core Knowledge resources
