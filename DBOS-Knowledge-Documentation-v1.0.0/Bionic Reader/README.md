# Bionic Reader

## Overview

Bionic Reader converts supplied text into Bionic Reading format by partially bolding the beginning of each word to create visual fixation points.

## Purpose

Support easier visual processing of text while preserving the original content exactly.

## Use This Package When

- text needs to be converted into Bionic Reading format;
- the original wording and structure must remain unchanged.

## Inputs

- text supplied by the user.

## Outputs

- clean Markdown containing the original text with only the required portions of words bolded.

## Core Rules

- Do not rewrite, correct, paraphrase or summarise.
- Preserve wording, punctuation, headings, paragraph spacing and line breaks.
- Apply bold formatting only to the defined opening portion of each word.
- Return only the converted text.
- Do not add commentary or explanation.
